   ```
   git clone https://github.com/airdropinsiders/Gaea-Bot.git && cd Gaea-Bot
   ```
2. Run
   ```
   pip install -r requirements.txt
   ```

   ```
   nano data.txt
   ```
3. Run Bot
   ```
   python bot.py
   ```
